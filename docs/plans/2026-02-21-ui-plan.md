# Huellas Humanitarias UI Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Build the visual foundation of the Huellas Humanitarias platform, including a premium stunning landing page, navigation, and a placeholder catalog. This must feature modern design aesthetics (glassmorphism, vibrant but harmonious colors, micro-animations) as required by our core principles.

**Architecture:** Next.js App Router (React Server Components where possible, Client Components for interactivity). Styling via Tailwind CSS with a custom configuration.

**Tech Stack:** Next.js, React, Tailwind CSS, Lucide React (for icons).

## Bite-Sized Tasks (Execution Plan)

### Task 1: Setup Design System & Layout Wrapper

**Files:**
- Modify: `tailwind.config.ts`, `globals.css`, `layout.tsx`
- Install: `lucide-react` (for icons)

**Step 1: Install dependencies**
Run: `npm install lucide-react`

**Step 2: Configure Tailwind and CSS**
Update `tailwind.config.ts` with brand colors (e.g., earthy tones representing humanity, solidarity, combined with modern sleek surfaces). Update `globals.css` with font imports (e.g., Inter or Outfit) and base layers.

**Step 3: Create base layout**
Modify `src/app/layout.tsx` to include the standard HTML structure and prepare valid slots for Navbar and Footer.

**Step 4: Commit**
```bash
git add src/app/tailwind.config.ts src/app/globals.css src/app/layout.tsx package.json package-lock.json
git commit -m "feat(ui): setup design system, fonts, and base layout"
```

### Task 2: Build Navigation and Footer

**Files:**
- Create: `src/components/layout/Navbar.tsx`
- Create: `src/components/layout/Footer.tsx`

**Step 1: Write Navbar**
Create a responsive Navbar with a glassmorphism effect (backdrop-blur). Include links for "Catálogo", "Nuestro Impacto", and a placeholder for "Iniciar Sesión" / Cart icon.

**Step 2: Write Footer**
Create a clean footer with simple links and a copyright notice.

**Step 3: Integrate into Layout**
Import and use Navbar and Footer inside `src/app/layout.tsx`.

**Step 4: Commit**
```bash
git add src/components/layout/ src/app/layout.tsx
git commit -m "feat(ui): build global navbar and footer"
```

### Task 3: Build the Landing Page (Hero & Impact Counter)

**Files:**
- Create: `src/components/home/HeroSection.tsx`
- Create: `src/components/home/ImpactCounter.tsx`
- Modify: `src/app/page.tsx`

**Step 1: Create Hero Section**
Build a compelling Hero Section with a strong value proposition: "Compra 2, Donamos 1". Add a Call-to-Action button to "Ver Colección". Use subtle animations.

**Step 2: Create Impact Counter**
Build a component that visually displays the number of shoes donated so far (mocked to e.g., "1,240 pares donados").

**Step 3: Assemble Home Page**
Update `src/app/page.tsx` to compose `HeroSection` and `ImpactCounter`.

**Step 4: Commit**
```bash
git add src/components/home/ src/app/page.tsx
git commit -m "feat(ui): build landing page with hero and impact counter"
```
